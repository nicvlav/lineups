# Lineups

**Auto-balanced soccer team generator with player stats**

[lineups.nicolasvlavianos.workers.dev](https://lineups.nicolasvlavianos.workers.dev)

Currently in limited testing with a small group.

## Core Features

**Auto-balance teams** based on player stats (attack, defense, athleticism) with configurable zone weights

**Player stats system** - persistent player data with detailed performance metrics

**Click-to-swap** player positioning (no freeform dragging)

**Formations** - preset tactical setups for soccer

## Tech Stack

React + Vite • Supabase (PostgreSQL) • Cloudflare Workers • Bun + Biome • TypeScript
